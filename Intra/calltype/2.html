911 Open Lines, much the same as hangups, can come in for a wide range of situations. Anything from pocket change pressing the emergency button on a phone without the owner's knowledge, to the victim in a domestic violence situation calling for help without being able to talk out loud. Due to the massive uncertainty we must give these call types the respect of an obvious emergency.

911 Open Lines can also come in as a child playing on the phone. In these situations we may have to use more simplified questioning techniques. The goal is to get the phone to an adult. If no adult is present we need to attempt to confirm the address with the child. Looking for a piece of mail with an address is an ideal place to start. Cross reference this with RapidSOS and Mapping if possible.

911 Calls can be made from older disconnected cell phones. Often we get calls from children playing with an older phone. We also get legitimate calls for service via this method on occasion, for example if a caller has no minutes on a prepaid plan. This makes it difficult in the event we would need to recontact the RP. We should try to get a number for a functioning phone on scene if possible.

We still want to answer all of our standard questions for these call types if we are able to. If you have some indication that the caller is in a dangerous situation, there are many steps we can take to get a non-verbal image of the scene. We can instruct them that if they have to break off communications to turn on speaker phone so we can listen to any exchanges taking place.

Dispatch: if you can hear me press a button. If you are not able to talk press a button. We're going to do one button press for yes, two for no. Do you understand? I have you at such and such address. Is this correct.

WHO is the RP?
WHERE is the RP calling from?
DO you need to report an emergency?
--If yes, immediately shunt to the appropriate call type on the left hand side, or to EMD
--If no, continue:
IF this was an emergency would they be comfortable telling us?
